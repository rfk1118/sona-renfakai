                                                                                                                                                                                                        # 导读

                                                                                                                                                                                                        # 基础

                                                                                                                                                                                                        # 图类型

                                                                                                                                                                                                        • 无向图
                                                                                                                                                                                                          • 图中的边没有方向，只处理连接问题
                                                                                                                                                                                                        • 有向图
                                                                                                                                                                                                          • 图中的边有方向

                                                                                                                                                                                                        # 图权重

                                                                                                                                                                                                        • 无权图
                                                                                                                                                                                                          • 所有权重相等
                                                                                                                                                                                                        • 有权图
                                                                                                                                                                                                          • 每条边都有各自权重，后面会引出连接花费，最短路径问题等

                                                                                                                                                                                                        # 搜索

                                                                                                                                                                                                        • DFS(深度优先搜索)
                                                                                                                                                                                                          • 使用栈作为辅助内存空间，一条路走到黑，在返回机制
                                                                                                                                                                                                        • BFS(广度优先搜索)
                                                                                                                                                                                                          • 使用队列作为辅助内存空间，就近原则处理

                                                                                                                                                                                                        # 最小生成树

                                                                                                                                                                                                        • Kruskal’s algorithm
                                                                                                                                                                                                        • Prim’s algorithm

                                                                                                                                                                                                        # 最短路径

                                                                                                                                                                                                        • Floyd-Warshall Algorithm
                                                                                                                                                                                                          • 解决所有顶点间的最短路径问题
                                                                                                                                                                                                        • Dijkstra’s Algorithm
                                                                                                                                                                                                          • 解决单源最短路径问题
