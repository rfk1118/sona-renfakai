## 为什么写博客？

工作多年发现能清晰、准确、简单的表达想法是一件很难的事，写作恰巧能锻炼这项能力，并且这种方式很高效，拥有这项能力可以让沟通、阅读成本更低。

学习技术看概念三天就忘了，在写作期间发现了适合自己的学习方案（读源码、找理论知识、写博客、讲解），坚持很久后发现学技术变的简单了。
